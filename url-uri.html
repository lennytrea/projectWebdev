URLs, URIs & URNs

What is a URI?

A Uniform Resource Identifier, any string of characters that identifies a resource on the internet.
URLs and URNs are both types of URI.

What is a URL?

A Uniform Resource Locator, It identifies a resource and tells you how to find it.

Anatomy of a URL

  • Scheme - The protocol used. Example: https://
  • Domain - The website name. Example: www.google.com
  • Path - The specific page or file. Example: /search
  • Query - Extra parameters. Example: ?q=dns
  • Full example: https://www.google.com/search?q=dns

What is a URN?

A Uniform Resource Name. Identifies a resource by name rather than location.

Quick Comparison

  • URI - Identifies a resource.
  • URL - Identifies AND locates a resource, typed in a browser.
  • URN - Identifies a resource by a permanent name. Location independent.
