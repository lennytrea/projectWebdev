World Wide Web (WWW)

What is the World Wide Web?

A system of interlinked documents and resources accessed via the internet. It is different from the internet itself.

Web vs. Internet

  • The Internet is the global network of connected devices — the infrastructure.
  • The Web is a service that runs on top of that infrastructure.
  • Email, online gaming, and video calls use the internet but are not the web.

Key Facts

  • First website published on 6 August 1991.
  • Built on three core technologies: HTML, HTTP, and URLs.

How It Works

  • Web pages are written in HTML and stored on web servers.
  • You request a page via its URL in a browser.
  • The browser fetches it using HTTP/HTTPS.
  • The page is displayed and hyperlinks connect it to other pages.

Types of Web Content

  • Static - Fixed content that is the same for every visitor.
  • Dynamic - Content that changes based on user interaction or data.
