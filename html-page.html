Hypertext Markup Language (HTML)

What is HTML?

HyperText Markup Language is the standard language used to create and structure content on the web. Every web page you visit is built on HTML.

Key Facts

  • Created by Tim Berners-Lee in 1991.
  • Not a programming language — it is a markup languagetelling the browser what content to display and how to structure it.

How It Works

  • HTML uses tags to label and structure content.
  • Tags are written in angle brackets — e.g. <h1>, <p>, <img>.
  • The browser reads the HTML and renders it as a visible web page.

Common HTML Tags

  • <h1> to <h6> - Headings from largest to smallest.
  • <p> - A paragraph of text.
  • <a> - A hyperlink to another page or resource.
  • <img> - Displays an image.
  • <ul> / <ol> - Unordered and ordered lists.
  • <div> - A container used to group and layout content.

HTML, CSS & JavaScript

  • HTML - Provides the structure and content of the page.
  • CSS - Controls the visual styling and layout.
  • JavaScript - Adds interactivity and dynamic behaviour.
  • All three work together to build modern web pages.
